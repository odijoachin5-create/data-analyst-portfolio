📊 Projet  — Superstore Sales Star Schema


Construction d'un modèle en étoile (star schema) à partir du dataset Sample Superstore : 1 table de faits (Facts_Sales) reliée à 3 tables de dimensions (Dim_Customer, Dim_Product, Dim_Geo) et une table de dates dédiée. Implémentation de la Time Intelligence (TOTALYTD, SAMEPERIODLASTYEAR) pour comparer les ventes année sur année, mise en forme conditionnelle (vert/rouge) selon la profitabilité par sous-catégorie, et navigation par drillthrough vers une page de détail produit.


Built a star schema model from the Sample Superstore dataset: 1 fact table (Facts_Sales) connected to 3 dimension tables (Dim_Customer, Dim_Product, Dim_Geo) plus a dedicated date table. Implemented Time Intelligence (TOTALYTD, SAMEPERIODLASTYEAR) for year-over-year sales comparison, conditional formatting (green/red) based on sub-category profitability, and drillthrough navigation to a product detail page.

Skills: Star Schema Modeling, DAX Time Intelligence, Conditional Formatting, Drillthrough, Data Relationships
