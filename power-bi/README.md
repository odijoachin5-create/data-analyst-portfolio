# 📊 Power BI Portfolio

4 projets démontrant la maîtrise des compétences PL-300 : Power Query, modélisation star schema, DAX, et visualisation interactive.

| Projet | Description | Compétences clés |
|--------|-------------|-------------------|
| [Titanic Survival Analysis](./titanic-survival) | Analyse exploratoire, taux de survie par sexe/âge/classe | Power Query, DAX (CALCULATE, SWITCH), nettoyage de données |
| [Superstore Star Schema](./superstore-star-schema) | Modèle en étoile, Time Intelligence, drillthrough | Star schema, relations, TOTALYTD, conditional formatting |
| [Financial Dashboard](./financial-dashboard) | KPI financiers, analyse de rentabilité par segment/pays | Mesures DAX, KPI cards, business insights |
| [HR Attrition Analysis](./hr-attrition) | Analyse RH, identification des départements à risque | DAX (CALCULATE, DIVIDE), workforce analytics |

Chaque dossier contient le fichier `.pbix` source et une description détaillée du projet.
