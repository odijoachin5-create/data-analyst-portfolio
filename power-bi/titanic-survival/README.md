📊 Projet  — Titanic Survival Analysis

Analyse exploratoire du dataset Titanic dans Power BI : nettoyage des données (gestion des valeurs manquantes pour Age, Cabin, Embarked), création de mesures DAX (taux de survie, CALCULATE, SWITCH pour le regroupement par tranches d'âge), et construction d'un rapport interactif sur 2 pages avec slicers dynamiques. L'analyse révèle que le taux de survie des femmes (74%) était nettement supérieur à celui des hommes (19%), et que les enfants avaient le meilleur taux de survie parmi toutes les tranches d'âge.

Exploratory analysis of the Titanic dataset in Power BI: data cleaning (handling missing values for Age, Cabin, Embarked), DAX measures (survival rate, CALCULATE, SWITCH for age bucketing), and a 2-page interactive report with dynamic slicers. The analysis reveals women had a 74% survival rate vs 19% for men, and children had the highest survival rate across all age groups.

Skills: Power Query, DAX (SUM, CALCULATE, SWITCH, VAR), Data Cleaning, Interactive Visuals
